Responsive Web Design - Lab 3

This page demonstrates a Bootstrap layout for Lab 3.

Using bootstrap allows us to create responsive and visually appealing web pages with minimal effort. The grid system and pre-built components make it easy to design layouts that work well for different screen sizes such as phones, tablets, and desktops. By utilizing Bootstrap's classes and features, we can ensure that our web page looks great and functions properly across various devices. It also has the bennefits mentioned below:

  • Responsive layout
  • Mobile Friendly design
  • Bootstrap Grid System
